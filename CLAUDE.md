# Claude Code 使用ガイドライン

## 言語設定

このプロジェクトでは、Claude Code との対話において以下の言語設定を使用します：

- **入力言語**: 日本語
- **出力言語**: 日本語

## 重要事項

Claude Code を使用する際は、以下の点に注意してください：

1. すべてのユーザー入力は日本語で行います
2. Claude からの応答もすべて日本語で提供されます
3. コード内のコメントも日本語で記述します
4. ドキュメント、仕様書、説明文もすべて日本語で記述します

## コミュニケーション方針

- 技術的な正確性を優先しつつ、わかりやすい日本語で説明します
- コード例やエラーメッセージは必要に応じて日本語で補足説明を加えます
- 専門用語は適切に日本語訳を使用し、必要に応じて英語の原語も併記します
